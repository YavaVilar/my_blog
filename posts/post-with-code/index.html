Librarie OlympicsWeather

Le package olympicsWeather est une librairie de R dédiée à la récupération de prévisions météorologiques en temps réel pour les différents sites olympiques qui aurant lieu du 26 juillet au 11 août 2024 pour célébrer les Jeux Olympiques 2024 à Paris. Il offre des fonctionnalités permettant d’obtenir des données détaillées sur les conditions météorologiques horaires des 7 prochains jours.

Installation du Package

Pour installer le package olympicsWeather depuis GitHub, vous pouvez utiliser le package remotes. Il est nécessaire d’avoir installé le package remotes au préalable. Il peut être installé avec la commande suivante :

install.packages("remotes")

Ensuite, on peut utiliser la commande suivante pour installer olympicsWeather :

remotes::install_github("YavaVilar/olympicsWeather")

La librarie peut être importée de la façon suivante, ce qui permet d’utiliser ses fonctions pour avoir accès aux données météorologiques.

library(olympicsWeather)

La fonction get_forecast est la fonction principale de la librairie. Elle retourne un dataframe avec 5 colonnes relatives à la date et heure, la température, la température ressentie, la probabilité de précipitation et la précipitation effecitve. Ci-dessous, on trouve trois exemples d’implémentation pour le stade de France, le Tahiti et les coordonées géographiques 43.276703, 5.334791. Des graphiques ont été effectués pour visualiser les données sur la période 04/03/2024-10 - 10/03/2024.
