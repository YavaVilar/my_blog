Librarie OlympicsWeather

Le package olympicsWeather est une bibliothèque R dédiée à la récupération de prévisions météorologiques en temps réel pour différents sites olympiques existants. Il offre des fonctionnalités permettant d’obtenir des données détaillées sur la température, la température ressentie, la probabilité de précipitations et les précipitations effectives.

Installation du Package

Pour installer le package olympicsWeather depuis GitHub, vous pouvez utiliser le package remotes. Assurez-vous d’avoir le package remotes installé au préalable. Si ce n’est pas le cas, vous pouvez l’installer avec la commande suivante :

install.packages("remotes")
The following package(s) will be installed:
- remotes [2.4.2.1]
These packages will be installed into "~/Desktop/my_blog/renv/library/R-4.3/x86_64-w64-mingw32".

# Installing packages --------------------------------------------------------

Une fois que le package remotes est installé, vous pouvez utiliser la commande suivante pour installer olympicsWeather :

remotes::install_github("YavaVilar/olympicsWeather")
